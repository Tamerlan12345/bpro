## Формат записей:
## ГГГГ-ММ-ДД — [Тип бага] **Проблема:** [Что ломалось] **Решение:** [Как исправили навсегда]

## 2026-02-25 — [Безопасность/IDOR] **Проблема:** Пользователи с ролью 'user' могли просматривать и изменять данные чатов (версии, комментарии, транскрипции), которые им не принадлежат, зная ID чата. Проверка прав доступа отсутствовала в API эндпоинтах. **Решение:** Добавлена функция `checkChatAccess` и проверка владения чатом во все уязвимые эндпоинты (`/api/chats/:id/*`). Написан регрессионный тест `backend/tests/security_idor.test.js`.
